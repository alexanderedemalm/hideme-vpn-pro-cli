## ABOUT

The hideme-vpn script simplifies interaction with the underlying \
hideme Linux CLI by providing a clean, user-friendly, and interactive \
menu system. This allows users to easily manage connections, check \
service status, and perform common utility functions without \
memorizing complex command-line flags.                                      


### üöÄ Features

- Interactive Menu: Navigate through service management, connection \
setup, and utility functions using keyboard input.

- Service Control: Easily start, stop, restart, and check the status \
of the hideme background service.

- Quick Connect: Simplified options for connecting and disconnecting.

- Status Overview: Retrieve detailed connection information and IP- \
addresses.
